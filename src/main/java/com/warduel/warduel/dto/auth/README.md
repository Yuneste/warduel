# Authentication DTO Package

This package contains Data Transfer Objects for authentication and user management.

## Planned DTOs:

### Authentication
- **RegisterRequest** - User registration payload
- **LoginRequest** - Login credentials
- **AuthResponse** - JWT token response
- **GuestCreateRequest** - Anonymous user creation

### User Management
- **UserDTO** - User profile data
- **UpdateProfileRequest** - Profile update payload

## Purpose:
- Input validation
- Request/Response mapping
- API contract definition

## Status: **Ready for implementation**
