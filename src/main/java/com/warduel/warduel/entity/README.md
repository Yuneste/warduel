# Entity Package

This package contains JPA Entity classes for database persistence.

## Planned Entities:

### User Management
- **User.java** - User accounts (registered & guest users)
- **UserStats.java** - Aggregate user statistics
- **Friendship.java** - Friend relationships

### Game System
- **GameMode.java** - Game mode configurations
- **PersistedGameSession.java** - Completed game records
- **MatchInvite.java** - Friend invite system
