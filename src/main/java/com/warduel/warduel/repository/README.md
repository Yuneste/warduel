# Repository Package

This package contains Spring Data JPA Repository interfaces for database access.

## Planned Repositories:

- **UserRepository** - User CRUD operations and queries
- **UserStatsRepository** - Statistics queries
- **FriendshipRepository** - Friend relationship queries
- **GameModeRepository** - Game mode lookup
- **PersistedGameSessionRepository** - Game history queries
- **MatchInviteRepository** - Invite code management

## Features:
- Custom query methods
- Pagination support
- Aggregate statistics
- Optimized lookups
